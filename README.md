# CS499 Game Launcher

## Overview
The CS499 Game Launcher is a full-stack desktop-style web application that allows users to launch games from a centralized interface. Built with a modern React frontend and a lightweight SQLite database, the application provides persistent storage, fast performance, and a clean user experience.

This project demonstrates full-stack development, local data persistence, and system-level interaction through a modern web tech stack.

---

## Features

- **Game Library Management (Admin only)**
  - Add, edit, and remove games o
  - Store executable paths and metadata

- **One-Click Launching**
  - In the future would handle local system execution, for now just downloads the game zip onto desktop.

- **Persistent Storage**
  - Uses SQLite to store game data
  - Automatically loads saved library on startup

- **Modern UI**
  - Built with React and styled using Tailwind CSS
  - Responsive and clean design

- **Fast Development Environment**
  - Powered by Vite for fast builds and hot module reload (HMR)

---

## Tech Stack

- **Frontend:** React (JavaScript)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Database:** SQLite
- **Runtime:** Node.js
